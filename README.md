# Shift Mon

An open source monitoring and logging tool based on Telegraf, Victoriametrics, Loki, Grafana, and Uptime Kuma.

## Security Features
* LDAP for Grafana
* User defined OIDC sign on for Grafana
* Google SSO (OIDC)
* Updates Every time the service is restarted
